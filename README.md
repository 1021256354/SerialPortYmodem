# SerialPortYmodem

串口通过YMODEM协议进行文件传输。程序中涉及到YMODEM协议知识，详细介绍见维基百科[YMODEM](https://en.wikipedia.org/wiki/YMODEM)。

## 开发环境

* 框架：Qt 5.7.1
* 编译器：MSVC2015_64bit
* IDE：Qt Creator 4.2.0 社区版
* 操作系统：Windows 10 专业版
